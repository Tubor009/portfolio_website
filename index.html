My technical skills include:

MS Excel

I'm experienced with Excel data cleaning functions (e.g. CONCATENATE, TEXTJOIN, LEFT, RIGHT, MID, TRIM, CLEAN, SUBSTITUTE); relative, mixed & absolute cell references; named ranges; tables; logical functions (e.g. IF, nested IF, AND, OR, SUMIF); lookup functions (VLOOKUP, XLOOKUP, INDEX & MATCH); data validation; conditional formatting; macros; visualizations with charts, sparklines and shapes; Pivot Tables and Charts; interactive dashboards; Power Query; Power Pivot and Data Modelling.

Google Sheets

I am learning to appreciate all the new features Google Sheets has to offer, including real-time collaboration with other users, connectivity to Google BigQuery, ability to link with Google Slides and update visualizations with one click, version control, connecting to external data sources (e.g. GOOGLEFINANCE, GOOGLETRANSLATE), ability to easily share files, automatic saving, ability to set notifications, real-time chat feature, free access to files from any computer.

SQL

I am proficient in SQL. I'm constantly learning new tools, however I'm experienced with data exploration and data cleaning using: basic queries, ordering, grouping, filtering; aggregate functions (e.g. MIN and MAX, AVG, COUNT, SUM), functions including CAST, CONVERT, UPDATE, SUBSTRING, CHARINDEX, LEN, PARSENAME; CASE statements; ROW_NUMBER; Window functions; ALTER TABLE; DROP COLUMN; Joins; CTEs; Temp Tables; creating views; converting data types.

Tableau

I have experience with Tableau because it's a great visualization tool. I've worked with different data sources and data types: numerical, string, date, geographical, Boolean. Utilizing dataset live connections and extracts. Using inner, left, right, outer and union joins. I am comfortable using dimensions and measures, shelves, cards and filters, creating calculated fields. Created worksheets and dashboards for multiple projects, while focusing on the proper dashboard structure.

R Programming

I use RStudio and R for data analysis of large datasets. I am familiar with the syntax, data types and data structure. I'm comfortable using assignment, arithmetic, relational and logical operators. I have worked with data frames, data cleaning, transformational and organizational functions. I've used visualization packages, e.g. ggplot2 and plotly.
